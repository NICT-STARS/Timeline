このサンプルは以下の仕様で実装されております。
  • 通常モード
    • 日時は10分刻みで表示されます。
    • ボタン、または ボタンで10分毎に日時を移動します。
    • ボタン、または ボタンで24時間毎に日時を移動します。
    • 画面上部の日時をクリックするとカレンダが表示され日付の移動が出来ます。
    • 画面下部の各日時ラベルの辺り、または をダブルクリックすると時間軸バーがズームアウト（右クリック）・ズームイン（左クリック）します。マウスホイールやピンチ操作（タッチデバイスの場合）でもズームイン・アウトします。
    • 時間軸バーのズームアウトは、1日間 → 2日間 → 3日間 → 4日間 と4段階にスケールが切り替わります。ズームインは、その逆方向に切り替わります。
    • ボタン、または ボタンを長押しすると自動再生モードになります。
  • 自動再生モード
    • 再生期間を表す帯 が表示されます。左右には開始日時と終了日時が表示されます。
    • 再生期間の帯 は、左右の端をドラッグすると開始日時または終了日時が移動します。中央辺りをドラッグすると帯全体が左右に移動します。
    • 再生期間の帯 の左右に表示される開始日時または終了日時をクリックすると、現在日時がその日時に移動します。
    • をクリックすると繰り返し再生のon/offが切り替わります。
    • をクリックすると再生速度（fps）が切り替わります。
    • 再生速度（fps）は、1 ～ 10 の10段階にトグルで切り替わります。
    • ボタンと ボタンの下に再生時間間隔が表示されます。
    • ボタン、または ボタンをクリックすると再生時間間隔が切り替わります。
    • 再生時間間隔は、10分 → 30分 → 1時間 → 3時間 → 24時間 と5段階にトグルで切り替わります。
    • ボタン、または ボタンで再生時間間隔毎に日時を移動します。
    • ボタン、または ボタンを長押しすると自動再生が開始します。
    • 自動再生中は が表示され、クリックすると再生速度（fps）が切り替わります。
    • 自動再生中は画面上のいずれか（ 以外の場所）をクリックすると自動再生が停止します。
    • この説明文章中の、いずれかをクリックすると通常モードに戻ります。
